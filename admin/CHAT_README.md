# كيف تُعلّم "عزم" بسرعة (دليل بسيط)

## 1) أول تشغيل
- شغّل Qdrant (مرة واحدة):
  - `docker run -p 6333:6333 -p 6334:6334 -v qdrant_data:/qdrant/storage qdrant/qdrant:latest`
- افتح إعدادات عزم: لوحة التحكم > عزم – الذكاء > إعدادات عزم
  - جرّب “اختبار OpenAI” و“اختبار Qdrant”.
  - إن ظهر النجاح، أنت جاهز.

## 2) إدخال معرفة فوري (للBREف)
- ادخل على: لوحة التحكم > عزم – الذكاء > مصادر المعرفة
- الصق مقاطع قصيرة وواضحة (كل سطر مقطع).
- اضغط “إدخال إلى RAG”.
- اختبر في “عزم – المساعد” بسؤال مرتبط، وتأكد أن “المصدر” يظهر أسفل الرد.

نصيحة: اجعل كل مقطع فكرة واحدة، ليسهل على "عزم" استرجاعها.

## 3) إدارة واعتماد (للعمل اليومي)
- الإحصاءات: عزم – الذكاء > إحصاءات عزم
  - سترى أكثر الأسئلة تكرارًا وما يحتاج تعليم.
  - اضغط “أرسل للتعليم” لإضافة السؤال لقائمة الانتظار.
- قائمة الانتظار: عزم – الذكاء > قائمة الانتظار
  - اعتمد/ارفض البنود. المعتمَد يُضاف للمعرفة.

## 4) استيراد/تصدير
- عزم – الذكاء > تصدير/استيراد
  - نزّل JSON للأسئلة.
  - استورد JSON بمقاطع ومعها بياناتها (lang/category/title).

## 5) فهرسة الموقع (اختياري)
- من إعدادات عزم: شغّل “زحف كامل” أو “زحف سريع”.
- يقرأ صفحات موقعك ويضيفها كمعرفة، مع ضبط الحد الأقصى والعمق.

هذا كل شيء. عندما تُدخل سياسة أو معلومة جديدة، اختبرها مباشرة في الشات. إذا لم يجد “عزم” مصدرًا، أرسِل السؤال للتعليم من صفحة الإحصاءات.
